# 🌸 Cordillera – Flower E-commerce Website

**Cordillera** is a visually engaging flower e-commerce front-end project. Built with HTML, CSS, and SwiperJS, it showcases floral arrangements, promotional packs, and interactive UI elements in a responsive, modern layout. The design emphasizes elegance and accessibility, with a focus on a smooth user experience for a flower delivery business or online florist.

---

## Features

- Responsive and clean design  
- Flower product showcase with images and descriptions  
- Contact form for customer inquiries  
- Modern layout with easy navigation
