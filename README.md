🌸 Cordillera – Flower E-commerce Website
Cordillera is a visually engaging flower e-commerce front-end project. Built with HTML, CSS, and SwiperJS, it showcases floral arrangements, promotional packs, and interactive UI elements in a responsive, modern layout. The design emphasizes elegance and accessibility, with a focus on a smooth user experience for a flower delivery business or online florist.

🌐 Demo
You can view the live version here.

📌 Features
🌺 Hero section with promotional messaging and call to action

📷 SwiperJS image slider showcasing featured floral arrangements

🛍️ Product cards with hover effects, discount overlays, and rating stars

📱 Responsive design with mobile-friendly navigation

❓ FAQ accordion section to assist users

🎨 Clean and modern aesthetic using Normalize.css and custom styles

🛠️ Technologies Used
HTML5 – Semantic and accessible structure

CSS3 – Custom styling and layout

SwiperJS – Image slider/carousel functionality

Bootstrap Icons – Iconography

Normalize.css – Cross-browser CSS normalization
