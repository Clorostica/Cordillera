🌸 Cordillera – Flower E-commerce Website
Cordillera is a visually engaging flower e-commerce front-end project. Built with HTML, CSS, and SwiperJS, it showcases floral arrangements, promotional packs, and interactive UI elements within a responsive, modern layout. The design emphasizes elegance and accessibility, focusing on a smooth user experience for flower delivery businesses or online florists.

✨ Features
Responsive and clean design: Works beautifully on all devices, from mobile phones to desktops.

Product showcase: High-quality images and detailed descriptions of flower arrangements and promotional packs.

Interactive UI: Smooth transitions and swiper sliders for an engaging browsing experience.

Contact form: Allows customers to easily send inquiries or requests.

Easy navigation: Intuitive layout with accessible menus and buttons.
